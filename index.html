About

This website provides a blank canvas for playing with a browser console. There is no code loaded on this page that will cause any errors or change the default state of the Window.

However, if you would like to install some libraries to play with, you can do so by appending any of the terms as query parameters. Here are a list of the libraries supported:

  • jquery - jQuery 3.1.1
  • lodash - Lodash 4.16.6
  • moment - Moment 2.16.0
